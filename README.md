## Overview

This project simulates a full detection engineering pipeline across network telemetry, endpoint telemetry, SIEM correlation, investigation, and case management.

The lab integrates:

- Suricata
- Sysmon
- Wazuh
- Splunk
- TheHive

The objective is to validate how a single attack chain appears across multiple telemetry layers and how those signals can be correlated into a meaningful SOC workflow.

# Full-Detection-Pipeline-Lab
Cross-layer detection engineering pipeline using Suricata, Sysmon, Wazuh, Splunk, and TheHive to simulate real SOC detection workflows.
